# Portfolio UI Guide

## Visual Direction

- Use an editorial, magazine-like composition with generous breathing room.
- Keep the color system grounded: paper, ink, muted text, and one accent color.
- Favor tactile surfaces, soft borders, and restrained shadows over loud glassmorphism.
- Avoid layouts that look like a starter template or a generic AI-generated landing page.

## Section Order

1. Hero
2. Proof / stats
3. Featured work
4. Clients or collaborators
5. Testimonials
6. Contact

## Card Rules

- Make image size configurable through data or section metadata.
- Use consistent aspect ratios inside a section.
- Keep labels short and readable.
- Ensure links are obvious and can be changed without editing component structure.

## Tone

- Write direct, specific copy.
- Use concrete proof, not vague marketing language.
- Prefer a calm, senior, production-ready feel over trendy visual gimmicks.
